# Calculator Version One

## Overview
A basic calculator application built using Java. This is my first version, focusing on fundamental Object-Oriented Programming concepts.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Basic error handling

## Technologies Used
- Java (Core)
- NetBeans IDE
- Git for version control

## Project Structure
CalculatorV1/
├── src/
│   └── com/medie/calculator/
│       ├── Calculator.java
│       ├── Main.java
│       └── (other classes)
└── README.md



## Key Concepts Demonstrated
- Object-Oriented Programming (OOP)
- Composition
- Concrete classes
- Basic exception handling
- Method organization

## How to Run
1. Clone the repository
2. Open in NetBeans
3. Run Main.java

### Compiling from Command Line


javac src/com/medie/calculator/*.java
java src/com/medie/calculator/Main
